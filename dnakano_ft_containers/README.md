# ft_containers
Re-implementation of STL containters.

- list
- vector
- map
- stack
- queue

Only C++98 feature used and implemented.

# how to use
## use
Just include header files.

```
#include "list.hpp"
#include "vector.hpp"
#include "map.hpp"
#include "stack.hpp"
#include "queue.hpp"
```

## test
`make test` to run tester.
